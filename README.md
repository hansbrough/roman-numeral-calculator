# roman-numeral-calculator
Simple calculator that uses roman numerals as input. Operator precendence determined without use of `eval()` or an outside library.

Supports nested parenthesis for example:

```(I + II) * III```

or

```( C - V ) - ( IV + II )```

or even

```(I + ( V + X ))```

etc
